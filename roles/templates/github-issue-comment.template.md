# GitHub Issue 通用评论模板

```md
## [PACE] <步骤标识> - <状态>

- 角色：<角色名>
- 阶段：<阶段编号或 n/a>
- 当前步骤：<tracking-init(追踪初始化) | intake(接收) | discuss(讨论) | plan(计划) | execute(执行) | verify(验证) | archive(归档)>
- 已完成动作：
  - <这一阶段完成了什么>
- 当前结论：<当前阶段结论>
- 下一角色：<PACE-... | 无>
- 下一技能：<pace:... | 无>
- 相关产物：
  - <context / plan / run / verification / archive>
- 关键决策 / 变化：
  - <关键决策或范围变化>
- 风险 / 阻塞：
  - <没有则写 无>
- 下一步：
  - <下一角色或下一技能>
```
