# GitHub 初始化参数补充模板

```md
## 请补充初始化参数

当前不能继续接管，因为以下字段必须由你在本轮手动输入，不能猜测，也不能从 issue 正文、外部编排器参数或本地 `gh` / `git` 状态补齐：

- GitHub 用户（仓库 checkout / GitHub 访问使用的用户名）：<必填>
- 仓库地址（owner/repo）：<必填>
- 起始分支（从哪个分支开始修复）：<必填>
- Git Name：<必填>
- Git Email：<必填>

请直接按下面格式回复，不要省略字段：

github_user: <username>
repo: <owner/repo>
branch: <branch>
git_name: <name>
git_email: <email>
config_confirmed: <是|否>
```
