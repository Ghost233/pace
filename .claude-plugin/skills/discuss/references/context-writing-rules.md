# Context Writing Rules

`context.md` 必须回答：

- Goal
- Phase Boundary
- Locked Decisions
- Allowed Discretion
- Edge Cases / Failure Modes
- Rejected Alternatives
- Deferred Ideas
- References

规则：

- `Locked Decisions` 使用稳定编号，如 `D-01`
- `References` 只保留真正约束 planning / execution 的输入
- `discussion-log.md` 只做审计，不做下游真相源
