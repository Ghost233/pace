# Discussion Log Template

> 用途：为 `discussion-log.md` 提供统一结构。  
> 目标：保留 discuss 阶段的问答轨迹，方便人工回看。

## 模板

```md
# Phase <phase>: <title> Discussion Log

> 审计轨迹，仅供人工回看。  
> 以 `context.md` 作为下游真相源。

## Summary

- Areas discussed:
  - <area 1>
  - <area 2>
- Deferred:
  - <idea>

## <Area Name>

### Options Considered
- Option A: <描述>
- Option B: <描述>
- Option C: <描述>

### Selected
- <最终选择>

### Notes
- <补充说明>

### Rejected
- <被否决方案和原因>
```
