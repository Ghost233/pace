# Roadmap Template

## 模板

```md
# Roadmap

## Current Milestone

- <milestone 名称>

## Phases

### Phase 01: <title>
- Type: tech | requirement
- Owner Skill: <pace:map-codebase | pace:... | n/a>
- Expected Outputs:
  - <tech phase 必须列出文件或目录路径；requirement phase 写 n/a>
- Goal: <phase 完成后的结果>
- Requirements:
  - REQ-001
  - REQ-002
- Non-Goals:
  - <requirement phase 明确不做的内容；tech phase 写 n/a>
- Success Criteria:
  - <达到什么条件算 phase 完成；tech phase 写可验证结果>
- Entry Criteria:
  - <进入该 phase 前必须成立的条件>
- Done Criteria:
  - <该 phase 可进入下一步的硬条件>
- Depends On:
  - none
- Status:
  - planned
```
